What's busted, precious?

busted is a unit testing framework with a focus on being easy to use.

Clean interfaces, compatibility with most CI systems, and BDD-style syntax make using busted easy to write and easy to read. Options let you customize the output, from pretty terminal colors to JSON.

An extensible assert library allows you to extend and craft your own assert functions specific to your case with method chaining. 


require("busted")

describe("Busted unit testing framework", function()
  describe("should be awesome", function()
    it("should be easy to use", function()
      assert.truthy("Yup.")
    end)

    it("should have lots of features", function()
      assert.same({ table = "great"}, { table = "great" })
      assert.not.equal({ table = "great"}, { table = "great"})
    end)

    it("should be easy to read", function()
      local object_mock = mock(myObject)
      local spy = spy_on(object_mock.alert)

      object_mock.alert("Lua")

      assert.spy(spy).was_called_with("Lua")
    end)
  end)
end)
Get busted.

luarocks install busted

Don't have luarocks? Install luarocks or download busted from the Github repository and include it in your project.

Supported Asserts

The following functions are exposed under the asserts namespace and are chainable, for constructs like asserts.not.same({}, {}).

  • equal(...) checks if the objects are the same instance
  • same(...) checks if the objects contain the same data (useful for comparing tables)
  • not negates the expression (such as assert.not.equal)
